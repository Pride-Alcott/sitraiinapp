# Siemens Training Centre — Web App

React + Vite + Tailwind app for the Siemens Training Centre at Nelson Mandela University.
**All Base44 dependencies have been removed.** This project is fully self-hosted.

---

## Quick Start

### 1. Install dependencies
```bash
npm install
```

### 2. Install shadcn/ui components (required — one time only)
```bash
npx shadcn@latest init
```
When prompted, choose: **New York** style, **Neutral** base color, **Yes** to CSS variables.

Then add all required components:
```bash
npx shadcn@latest add button badge input textarea select label accordion sheet toast sonner
```

### 3. Run locally
```bash
npm run dev
```
Open http://localhost:5173

### 4. Build for production
```bash
npm run build
```
Output is in the `dist/` folder.

---

## Deployment

### Vercel (easiest)
1. Push this folder to a GitHub repo
2. Go to vercel.com → New Project → Import your repo
3. Vite is auto-detected — just click Deploy

### Netlify
1. Push to GitHub
2. netlify.com → New site → Import from GitHub
3. Build command: `npm run build`
4. Publish directory: `dist`

### Your own server (Nginx)
```nginx
server {
  listen 80;
  root /var/www/your-app/dist;
  index index.html;
  location / {
    try_files $uri $uri/ /index.html;
  }
}
```

## Making the Forms Send Real Emails

The enquiry and quote forms currently show a success message but don't send data anywhere.
Look for the `// TODO` comments in:
- `src/components/forms/EnquiryForm.jsx`
- `src/components/forms/CorporateQuoteForm.jsx`

**Option A — Formspree (no backend needed, free tier available)**
1. Sign up at formspree.io
2. Create a form and get your endpoint URL
3. Replace the TODO with:
```js
await fetch('https://formspree.io/f/YOUR_ID', {
  method: 'POST',
  headers: { 'Content-Type': 'application/json' },
  body: JSON.stringify(formData),
});
```

**Option B — EmailJS (sends email directly from frontend)**
1. Sign up at emailjs.com
2. `npm install @emailjs/browser`
3. Follow their React guide

**Option C — Your own API**
1. Add `VITE_API_URL=https://your-api.com` to `.env.local`
2. Replace the TODO with a fetch to your endpoint
